Artifact Overview

The artifact selected for this enhancement was originally created in CS-330 Computational Graphics and Visualization. The project focused on implementing graphical rendering concepts using OpenGL, including shader programs, vertex buffers, and object transformations.

The original artifact displayed simple colored geometry using basic rendering techniques. For this enhancement, the application was expanded to include advanced lighting, interactive camera controls, real-time transformations, depth testing, and improved rendering architecture.

Technologies Used:
  • OpenGL
  • GLFW
  • GLEW
  • GLM
  • stb_image
  • C++

Enhancement Goals

The primary goal of this enhancement was to improve both the graphical realism and the software engineering quality of the original artifact.

Enhancements Implemented

  • Implemented Phong lighting model
  • Added ambient, diffuse, and specular lighting
  • Integrated interactive camera controls
  • Added perspective projection and depth testing
  • Implemented orbiting light animation
  • Improved rendering organization and maintainability
  • Integrated additional graphics libraries
  • Enhanced debugging and shader management

Software Engineering Improvements

The enhancement process focused heavily on improving the maintainability, organization, and scalability of the original application.

Key Improvements

  • Separated rendering and shader responsibilities
  • Implemented reusable callback functions
  • Improved code readability and documentation
  • Added modular rendering logic
  • Improved camera and transformation management
  • Enhanced shader compilation and debugging support
Example Features Added:

• Dynamic lighting calculations
• Mouse and keyboard interaction
• Matrix transformations
• Perspective camera system
• Real-time object rendering

Reflection

Through this enhancement, I gained valuable experience in both graphics programming and software engineering refinement. One of the biggest lessons learned was the importance of modular and maintainable code organization when building increasingly complex graphical applications.

During development, I encountered several challenges related to lighting calculations, camera movement, shader debugging, and transformation matrices. Solving these issues improved my understanding of the graphics rendering pipeline and strengthened my debugging and problem-solving abilities.

This enhancement allowed me to transform a foundational graphics assignment into a more advanced interactive application while demonstrating software engineering skills related to maintainability, scalability, debugging, and refinement.
